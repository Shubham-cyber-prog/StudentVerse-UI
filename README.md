

# 🎓 StudentVerse UI

A student-driven open-source platform where learners build, share, and explore reusable UI components and useful mini tools — inspired by uiverse.io, built **by students, for students**.

---

## 🚀 Project Overview

**StudentVerse UI** is a collaborative platform that allows students to:
- Create beautiful UI components
- Share useful tools for daily student life
- Learn by contributing real-world projects
- Explore and reuse components instantly

This project is beginner-friendly, contribution-focused, and ideal for open-source programs like **GSSoC / Hacktoberfest**.

---

## ✨ Features

### 🎨 UI Components Gallery
- Buttons, Cards, Forms, Navbars, Loaders, Modals
- Live preview for every component
- Copy-ready code snippets
- Tags: HTML, CSS, JavaScript, Tailwind, Bootstrap

### 🛠️ Student Tools
- CGPA Calculator  
- Attendance Tracker  
- Pomodoro Timer  
- To-Do / Study Planner  
- Resume Builder (basic)

### 🤝 Contribution System
- Submit components with preview
- Contributor name & GitHub profile
- Beginner-friendly issue structure

### 🔍 Search & Filters
- Filter by category and tech stack
- Difficulty level: Beginner / Intermediate

### 🏆 Gamification (Upcoming)
- Contributor points & badges
- Leaderboard for top contributors

---

## 🧑‍💻 Tech Stack

- **Frontend**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Deployment**: Lovable
