## 📖 About

<img align="right" width="360" src="https://user-images.githubusercontent.com/74038190/212284100-561aa473-3905-4a80-b561-0d28506553ee.gif"/>

**StudentVerse UI** is a community-driven, open-source platform where students create, share, and explore beautiful **UI components** and **mini tools** built for everyday student life.

Inspired by [uiverse.io](https://uiverse.io) — but made **by students, for students**.

### 🎯 Our Mission

> *"Every student has something to build. StudentVerse UI gives them a stage to share it."*

- 📦 A growing library of **copy-paste ready** UI components
- 🛠️ Practical **student tools** — CGPA calc, Pomodoro, planners & more
- 🤝 A beginner-friendly space for **real-world open source** experience
- 🌍 Ideal for **GSSoC, Hacktoberfest**, and portfolio building

## ✨ Features

### 🎨 UI Components Gallery

A curated, ever-growing collection of components — all **live-preview** enabled and **copy-ready**.

| Category | Examples |
|---|---|
| 🔘 **Buttons** | Animated, gradient, glowing, 3D, icon buttons |
| 🃏 **Cards** | Profile cards, pricing cards, glassmorphism cards |

## 🚀 Getting Started

### Prerequisites

Make sure you have these ready:

| Tool | Version |
|---|---|
| Node.js | v18 or above |
| npm / bun | Latest |
| Git | Latest |

---

### Local Setup

**1. Fork & Clone the repo**
```bash
# Fork first (top-right Fork button), then:
git clone https://github.com/YOUR_USERNAME/studentverse-ui.git
cd studentverse-ui
```

**2. Install dependencies**
```bash
npm install
# or
bun install
```

**3. Start the development server**
```bash
npm run dev
# or
bun dev
```

**4. Open in browser**

```
http://localhost:5173
```

You're up and running! 🎉

---

### Available Scripts

```bash
npm run dev        # Start dev server
npm run build      # Build for production
npm run preview    # Preview production build
npm run lint       # Run ESLint checks
npm run test       # Run tests with Vitest
npm run format     # Format code with Prettier
```

## 🤝 How to Contribute

StudentVerse UI is **built for contributors**. Whether you're adding a new button style or building a full student tool — every contribution matters! 💜

### Ways to Contribute

| Type | Description |
|---|---|
| 🎨 **Add a UI Component** | Submit a new button, card, modal, navbar, etc. |
| 🛠️ **Build a Student Tool** | Add a useful tool for student daily life |
| 🐛 **Fix a Bug** | Find & fix issues in existing code |
| 📝 **Improve Docs** | Fix typos, add explanations, improve guides |
| 💡 **Suggest Features** | Open an issue with your idea |
| 🧪 **Write Tests** | Add Vitest unit/integration tests |

---

### 🔀 Contribution Steps

```bash
# Step 1: Fork this repo & clone your fork
git clone https://github.com/YOUR_USERNAME/studentverse-ui.git

# Step 2: Create a new branch
git checkout -b feat/your-component-name

# Step 3: Make your changes (see guidelines below)

# Step 4: Commit your work
git add .
git commit -m "feat: add glassmorphism login card"

# Step 5: Push to your fork
git push origin feat/your-component-name

# Step 6: Open a Pull Request on GitHub 🚀
```

---

### 🎨 Submitting a UI Component

To add a new component to the gallery:

1. Create your component file inside `src/components/gallery/`
2. Add a **live preview** using React
3. Include a clean **code snippet** (copyable)
4. Add metadata to `src/data/components.ts`:

```ts
{
  id: "glassmorphism-card",
  title: "Glassmorphism Card",
  category: "Cards",
  tags: ["CSS", "HTML"],
  difficulty: "Beginner",
  contributor: {
    name: "Your Name",
    github: "https://github.com/yourusername"
  },
  preview: GlassmorphismCard,   // React component
  code: `...your HTML/CSS here...`
}
```

5. Submit your PR — that's it! Your name goes on the card forever. 🏅

### ✍️ Commit Convention

We use [Conventional Commits](https://www.conventionalcommits.org/):

```
feat:     ✨ New component or tool
fix:      🐛 Bug fix
docs:     📝 Documentation update
style:    💄 CSS / UI-only change
refactor: ♻️  Code cleanup, no feature change
test:     🧪 Tests added
chore:    🔧 Config, build changes
```

## ❓ FAQ

<details>
<summary><b>Who can contribute to StudentVerse UI?</b></summary>
<br/>
Anyone! Whether you're a complete beginner learning HTML/CSS or an experienced React developer — there's always something for you here. We have issues labeled <code>good first issue</code> specifically for new contributors.
</details>

<details>
<summary><b>What kind of components can I submit?</b></summary>
<br/>
Any reusable UI component — buttons, cards, navbars, modals, loaders, forms, tooltips, badges, and more. Make sure it's original, visually appealing, and has clean code. Both HTML/CSS and React/Tailwind submissions are welcome!
</details>

<details>
<summary><b>Will my name be shown on my contribution?</b></summary>
<br/>
Yes! Every component you submit will display your name and link to your GitHub profile. Your contribution lives forever in the gallery and in the contributors section.
</details>

<details>
<summary><b>Does this count for GSSoC / Hacktoberfest?</b></summary>
<br/>
Yes! StudentVerse UI participates in GSSoC and Hacktoberfest. Contributions here count toward your program goals. Watch our repo and Discord for program-specific updates.
</details>

<details>
<summary><b>I found a bug. How do I report it?</b></summary>
<br/>
Open a GitHub Issue using the Bug Report template. Describe the problem, steps to reproduce, and screenshots if possible. We respond within 48 hours!
</details>
