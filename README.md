## 📣 About EduLinkUp

<img align="right" width="380" src="https://user-images.githubusercontent.com/74038190/229223263-cf2e4b07-2615-4f87-9c38-e37600f8381a.gif"/>

**EduLinkUp** is an open-source educational social networking platform built by students, for students. It brings together the best of academic networking, resource sharing, and collaborative learning — all under one roof.

Whether you're a **first-year fresher** looking for study partners or a **final-year student** mentoring juniors — EduLinkUp is your academic home on the internet.

### 💡 The Problem We Solve

- Students have **no dedicated platform** to network with peers and seniors
- Academic resources are **scattered** across Google Drive, WhatsApp groups, and Telegram channels
- Finding **mentors** or **project collaborators** is unnecessarily hard
- **Opportunities** like hackathons, internships & scholarships go unnoticed

> EduLinkUp solves all of this in one beautiful platform.

<br clear="right"/>

## ✨ Features

<details open>
<summary><b>👤 Student Profiles</b></summary>
<br/>

- Custom bio with **academic background**, branch & year
- Showcase **projects**, **skills**, and **certifications**
- Display your **contribution history** and activity
- Follow & connect with peers, seniors, and mentors
- Personalized **profile link** to share anywhere

</details>

<details open>
<summary><b>📰 Social Feed</b></summary>
<br/>

- Post academic updates, achievements, and resources
- **Like, comment, bookmark & reshare** with your network
- Rich text editor with image & file attachment support
- Tag posts by **subject, semester, branch, or topic**
- Trending content surfaced by the community

</details>

<details open>
<summary><b>📖 Study Rooms</b></summary>
<br/>

- Create public or private **subject-based study groups**
- Shared **resource library** — PDFs, links, notes, videos
- **Live discussion** via text channels (voice coming soon)
- Pinned announcements & weekly challenges
- Role-based access — Admin, Moderator, Member

</details>

<details open>
<summary><b>💡 Resource Hub</b></summary>
<br/>

- Upload and discover **notes, PYQs, PDFs, mind maps**
- **Upvote-based ranking** for quality content filtering
- Filter by university, semester, branch, subject
- Direct **download** with a single click
- Contributor credits on every resource

</details>

<details open>
<summary><b>🔔 Opportunity Board</b></summary>
<br/>

- Curated **internship & job listings** for students
- **Hackathon** and **competition** announcements
- **Scholarship** and fellowship alerts
- College fests, workshops & webinar listings
- One-click apply / register links

</details>

<details open>
<summary><b>🤖 AI Study Assistant <i>(Coming Soon)</i></b></summary>
<br/>

- Chat with your **uploaded notes & PDFs**
- Instant **document summarizer**
- Smart **quiz generator** from study material
- Personalized **revision reminders**
- Ask questions, get contextual answers

</details>

<details>
<summary><b>🏆 Gamification <i>(Coming Soon)</i></b></summary>
<br/>

- **XP points** for contributing resources, helping peers, and engaging
- Unlock **badges** — Top Contributor, Study Streak, Mentor, etc.
- **Monthly leaderboard** for most active students
- Institutional rankings — compete college-wide
- Profile **tier system** — Learner → Scholar → Legend

</details>

## 🚀 Getting Started

### ✅ Prerequisites

Ensure you have these installed before proceeding:

| Tool | Version | Install |
|---|---|---|
| Node.js | v18+ | [nodejs.org](https://nodejs.org) |
| npm / bun | Latest | Comes with Node / [bun.sh](https://bun.sh) |
| Git | Latest | [git-scm.com](https://git-scm.com) |

---

### ⚙️ Step-by-Step Setup

**Step 1: Fork & Clone**
```bash
# Fork this repo first (click Fork button above), then:
git clone https://github.com/YOUR_USERNAME/edulinkup.git
cd edulinkup
```

**Step 2: Install dependencies**
```bash
# Using npm
npm install

# OR using bun (faster)
bun install
```

**Step 3: Set up environment variables**
```bash
cp .env.example .env.local
```

Open `.env.local` and fill in your credentials:
```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_APP_NAME=EduLinkUp
```

**Step 4: Set up Supabase (Database)**
> Create a free project at [supabase.com](https://supabase.com), copy your URL and anon key, and paste them above. The tables will auto-configure on first run.

**Step 5: Start the development server**
```bash
npm run dev
# or
bun dev
```

🎉 Open [http://localhost:5173](http://localhost:5173) — you're live!
